# 苹果手机安装说明

## 安装网址

https://tyl114829-afk.github.io/fudan-study-planner/

应用已经具备 PWA 安装条件：手机布局、主屏幕图标、全屏模式和离线缓存均已配置。

## 发布后安装

1. 在 iPhone 上使用 Safari 打开应用网址。
2. 点击 Safari 底部的“分享”按钮。
3. 向下滑动，选择“添加到主屏幕”。
4. 名称保留“复旦备考”，点击右上角“添加”。

## 数据说明

- 进度默认保存在手机 Safari 本机；删除网站数据会丢失记录。
- 每周使用网页右上角“导出备份”，保存到 iCloud Drive。
- 电脑与手机不会自动同步。要实现实时同步，需要后续增加账号和云数据库。
- 原始丸子 PDF 不会上传到公开网站；手机应用内只包含原创整理的文化答题骨架。

## 隐私

GitHub Pages 只托管应用外壳和原创文化背诵骨架，不包含丸子PDF、浏览器数据或学习进度。
