# 背诵系统使用说明

## 核心方法

采用“主动回忆 + 扩展间隔”，复习节点为：

`D0 → D1 → D3 → D7 → D14 → D30 → D60 → D90`

- D0：第一次学完后立即合书回忆；
- D1：次日复习；
- D3：第一次学习后的第三天；
- 后续依次拉长。

研究依据：反复主动提取比反复阅读更有利于长期保持；分散学习比集中突击更稳定。节点不是神奇公式，真正关键是每次先回忆、后看书。

## 每个单元怎样背

1. 合上书，用5分钟口述或写关键词。
2. 打开原书/主资料，用3分钟核对遗漏与错误。
3. 再次合书，用2分钟重新完整表达。
4. 在界面中评级：
   - **不会**：核心结构无法说出，退一级，明天重来；
   - **模糊**：结构基本有但遗漏关键点，明天再复习；
   - **会了**：能完整、有序、带例子表达，进入下一个间隔。

“看着答案觉得熟”不能点会了。

## 单元应该多大

一个单元控制在10分钟内能完成一次回忆，例如：

- 合格：错误分析——步骤、贡献、局限与教学意义；
- 合格：声母的分类标准；
- 合格：文化适应假说；
- 不合格：整本《引论》；
- 不合格：现代汉语上册。

## 三天示例

### 第一天

- 新学“中介语特点”，加入背诵队列；
- 当天完成D0，点“会了”后安排到第二天。

### 第二天

- 先复习“中介语特点”D1；
- 再新学“错误分析”，完成D0；
- 今天的队列是旧单元1 + 新单元2。

### 第三天

- “中介语特点”尚未到D3时不重复刷；
- 复习“错误分析”D1；
- 新学“偏误来源”并完成D0。

随后系统会在相应日期自动把旧单元放回队列。每天不是把前面所有内容重背一遍，否则两周后必然崩盘。

## 在界面里怎么操作

- 学完当天任务，点击任务右侧 **◎**，将它加入背诵队列；
- 如果任务太大，点击“＋新单元”，拆成2—4个小单元；
- 每天21:10完成“今日到期背诵”；
- 完成后只按真实表现选择“不会/模糊/会了”；
- 数据自动保存在浏览器本机，定期使用“导出备份”。

## 滚动计划怎么用

- **智能顺延**：当天未完成任务进入次日；次日超过每日目标时长后，较新的、低优先级任务继续向后移动，直到每天重新落回容量以内。
- **旧任务优先**：系统保留任务最初日期。越早欠下的任务越不会被新任务挤走。
- **单项顺延**：任务右侧“→”只移动这一项，同时自动平衡后续日期。
- **锁定任务**：任务右侧“◇/◆”可锁定在当天，适合模考、报名和必须按日期完成的事项。
- **请假**：自动顺延当天全部未完成任务，并级联调整后续计划。
- **保底日**：状态很差时只保留约3.5小时最高优先级任务，其余内容顺延。
- **撤销**：滚动后发现不合适，可点击“撤销上次滚动”。刷新页面后撤销记录会失效。
- **未来7天负荷**：绿色正常、黄色接近上限、红色超载。智能平衡后通常不应出现红色。

## 作息与零基础计划

- 在“调整目标”中设置上午、下午、晚上各自的开始时间以及任务间休息时间；新生成的日期自动按这套作息排课。
- 当天临时改变作息，可直接点击任务卡片上的时间修改；或者点击“按我的时间重排”。
- 6月22日开始不做整套诊断：前两周先完成现汉、引论、英语语法长难句的基本术语和框架。
- 第一周容量约6小时，第二周约7小时，之后再进入成熟期容量，避免零基础第一周就用8—10小时计划制造假勤奋。
- 每个专业课单元固定执行“理解主资料→合书框架→少量测试”；英语先学语法和拆句，再按2010—2020逐篇精读。
- 7月15日开始政治；此前时间优先投入354、445和英语。
